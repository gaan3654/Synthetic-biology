Sukūrėme šią sistemą remdamosis Runge-Kutta metodu, diferencialinėms lygtims spręsti.

Sistemos naudojimas specifinėms funkcijoms.

  • Cosinuso lygtims spręsti įveskite: cos(lygtis)
  • Sinuso lygtims spręsti įveskite: sin(lygtis)
  • Kvadratiniai šakniai traukti įveskite: sqrt(lygtis)
  • Kubiniai šakniai traukti įveskite: cbrt(lygtis)
  • Lygties kelimui laipsniu įveskite: pagrindas^laipsnis
  • Natūrinaim logaritmui skaičiuoti įveskite: ln(lygtis)
  • Logarimtui skaičiuoti įveskite: log(pagrindas,logaritmuojamas reiškinys
  • Norint gauti tikslią skaičiaus Pi reikšmę įveskite: pi
  • Norint gauti tikslią skaičiaus E reikšmę įveskite: E
